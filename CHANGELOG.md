# Changelog

## 0.0.3

- Fix Windows playback when media duration is not available immediately after opening an audio file.

## 0.0.2

- Stop an in-progress start sound when the terminal command finishes.
- Play the result sound immediately after stopping the start sound for short commands.
- Replace the audio playback dependency with managed playback processes.

## 0.0.1

- Initial Terminal Hype Lines release.
- Add `mode1` and `mode_jujutsu` sound modes.
- Add start, success, warning, and error sound handling.
- Add explicit mode switching commands.
